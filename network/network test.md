# 📡 WiFi Channel Optimization (2.4GHz)

## 🎯 Objective
Analyze nearby WiFi networks and optimize the router channel to reduce interference and improve performance.

---

## 🧪 Tools Used
- Router Web Interface (TP-Link)
- WiFi Analyzer (Android)

---

## 📊 Initial Situation
- Frequency: 2.4GHz
- Channel: 1
- Signal Strength: ~ -50 dBm (strong signal)
- Observation: Multiple nearby networks overlapping on the same channel → interference

---

## 🔍 Analysis
Used WiFi Analyzer to scan nearby networks and visualize channel usage.

### Key Findings:
- Channel 1 → congested (multiple networks)
- Channel 6 → moderate usage
- Channel 11 → least interference (networks are weak and far)

---

## ⚙️ Action Taken
Changed router configuration:

- From: Channel 1
- To: Channel 11

---

## ✅ Result
- Reduced channel interference
- Improved WiFi stability
- Better performance in real-world usage

---

## 🧠 Key Concepts Learned
- WiFi operates on shared channels in 2.4GHz
- Only non-overlapping channels should be used: 1, 6, 11
- Signal strength is measured in dBm (closer to 0 = stronger)
- Channel congestion directly impacts performance
